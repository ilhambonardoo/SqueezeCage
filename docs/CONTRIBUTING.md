# Contributing

Terima kasih sudah berkontribusi! Berikut panduan singkat untuk berkontribusi ke proyek ini.

1. Proses kontribusi

- Fork repository atau buat branch baru di repo perusahaan.
- Gunakan branch per fitur: `feat/<deskripsi>` atau `fix/<deskripsi>`.
- Tulis commit message yang jelas.
- Buka Pull Request dengan deskripsi perubahan dan langkah untuk menguji.

2. Style guide

- TypeScript: gunakan tipe yang eksplisit ketika relevan.
- Styling: Tailwind CSS untuk styling UI.
- Format: jalankan formatter (prettier/eslint) sebelum commit.

3. Testing

- Tambahkan test untuk fitur baru jika memungkinkan.

4. Code review

- Pastikan tidak ada secret di commit.
- Sertakan screenshot atau langkah reproduksi untuk perubahan UI.

5. Branching & release

- Ikuti konvensi semantik untuk penamaan branch dan versi.

Jika ingin saya tambahkan template PR/ISSUE atau `CONTRIBUTING.md` yang lebih lengkap (dengan checklist PR, template commit), beri tahu saya.
